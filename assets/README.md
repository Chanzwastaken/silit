# SILIT Assets

This folder contains visual assets for the project.

## Current Assets

- **demo.gif** - 10-second demo showing SILIT in action

## Recommended Additional Assets

To make your README even better, consider adding:

### Screenshots
- `screenshot_main.png` - Main interface with hand detection
- `screenshot_word.png` - Word building demonstration
- `screenshot_training.png` - Training results/metrics

### Diagrams
- `architecture.png` - System architecture diagram
- `workflow.png` - User workflow diagram

### Branding
- `logo.png` - Project logo (512x512px)
- `banner.png` - Repository banner (1280x640px)

## How to Add More Assets

1. **Take Screenshots**
   - Run `python app/demo.py`
   - Use Snipping Tool or Print Screen
   - Save to this folder

2. **Create Diagrams**
   - Use draw.io, Figma, or similar
   - Export as PNG
   - Save to this folder

3. **Update README**
   - Add image references: `![Description](assets/filename.png)`
   - Keep images optimized (<500KB each)

## Image Guidelines

- **Format**: PNG for screenshots, GIF for animations
- **Size**: Optimize for web (<500KB)
- **Dimensions**: Max 1920px width
- **Quality**: High enough to be clear, but compressed

## Tools for Creating Assets

- **Screenshots**: Windows Snipping Tool, ShareX
- **GIFs**: ScreenToGif, LICEcap
- **Diagrams**: draw.io, Excalidraw
- **Editing**: GIMP, Paint.NET
- **Compression**: TinyPNG, ImageOptim
